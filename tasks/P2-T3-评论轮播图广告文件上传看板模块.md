```java
// 发表评论请求体
@Data
public class CreateCommentRequest {
    @NotNull
    private Long blogId;
    private Long parentId;       // null = 顶级评论
    @NotBlank(message = "评论内容不能为空")
    @Size(max = 500, message = "评论不能超过500字")
    private String content;
    private String nickname;     // 访客必填
    private String email;        // 访客必填
}

// 树形结构组装
public List<CommentVO> listComments(Long blogId) {
    List<TComment> all = commentMapper.selectList(
        new LambdaQueryWrapper<TComment>()
            .eq(TComment::getBlogId, blogId)
            .eq(TComment::getStatus, 1)   // 只返回已通过的
            .orderByAsc(TComment::getCreatedAt)
    );

    // 组装树形
    Map<Long, CommentVO> voMap = all.stream()
        .collect(Collectors.toMap(TComment::getId, CommentVO::from));

    List<CommentVO> roots = new ArrayList<>();
    for (TComment comment : all) {
        CommentVO vo = voMap.get(comment.getId());
        if (comment.getParentId() == null) {
            roots.add(vo);
        } else {
            CommentVO parent = voMap.get(comment.getParentId());
            if (parent != null) {
                parent.getChildren().add(vo);
            }
        }
    }
    return roots;
}
```

### 验收标准

- [ ] `GET /api/v1/comments/{blogId}` 返回树形结构，子评论在父评论的 `children` 字段中
- [ ] `POST /api/v1/comments` 新评论默认 `status=0`（待审核），前台不展示
- [ ] 后台 `PUT /admin/v1/comments/{id}/status` 审核通过后前台可见
- [ ] 删除父评论时，其子评论一并删除（级联处理）

```java
// 广告查询：过滤过期 + 按广告位
public List<AdvertisementVO> listAds(String position) {
    return adMapper.selectList(new LambdaQueryWrapper<TAdvertisement>()
        .eq(TAdvertisement::getStatus, 1)
        .eq(StringUtils.hasText(position), TAdvertisement::getPosition, position)
        .and(w -> w.isNull(TAdvertisement::getExpiredAt)
                   .or().gt(TAdvertisement::getExpiredAt, LocalDateTime.now()))
    ).stream().map(AdvertisementVO::from).collect(Collectors.toList());
}
```

## 模块三：文件上传（1d）

### 接口

```
POST /admin/v1/files/upload
Content-Type: multipart/form-data
```

### 实现（以阿里云 OSS 为例）

```java
@Service
@RequiredArgsConstructor
public class OssService {

    @Value("${voidpen.oss.endpoint}")
    private String endpoint;
    @Value("${voidpen.oss.bucket}")
    private String bucket;

    private final OSS ossClient;

    public String upload(MultipartFile file) {
        String originalName = file.getOriginalFilename();
        String ext = originalName.substring(originalName.lastIndexOf("."));
        // 按日期分目录，UUID 命名避免冲突
        String objectName = String.format("voidpen/%s/%s%s",
            LocalDate.now().toString().replace("-", "/"),
            UUID.randomUUID().toString().replace("-", ""), ext);

        try {
            ossClient.putObject(bucket, objectName,
                file.getInputStream(),
                new ObjectMetadata() {{ setContentType(file.getContentType()); }});
        } catch (IOException e) {
            throw new BusinessException(500, "文件上传失败");
        }

        return "https://" + bucket + "." + endpoint + "/" + objectName;
    }
}
```

**application-dev.yml 新增：**
```yaml
voidpen:
  oss:
    access-key-id: your-access-key
    access-key-secret: your-secret
    endpoint: oss-cn-hangzhou.aliyuncs.com
    bucket: voidpen-bucket
```

### 验收标准

- [ ] `POST /admin/v1/files/upload` 上传图片后返回可访问的 HTTPS URL
- [ ] 上传的图片在 OSS 控制台可见
- [ ] 超过 5MB 的文件返回错误提示（在 Controller 层校验）
- [ ] 同一文件名多次上传不会覆盖（UUID 命名保证唯一）

```java
// 今日访问量：从 Redis 汇总当天所有博客的浏览量增量
public long getTodayViews() {
    Set<String> keys = redisTemplate.keys(RedisKeys.BLOG_VIEWS + "*");
    if (CollectionUtils.isEmpty(keys)) return 0;
    return keys.stream()
        .mapToLong(key -> {
            Object val = redisTemplate.opsForValue().get(key);
            return val == null ? 0 : Long.parseLong(val.toString());
        }).sum();
}
```
