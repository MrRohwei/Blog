## 详细步骤

### 1. 数据看板（DashboardView.vue）

```vue
<template>
  <div class="dashboard">
    <!-- 统计卡片 -->
    <el-row :gutter="16" class="stats-row">
      <el-col :span="6" v-for="item in stats" :key="item.label">
        <el-card>
          <div class="stat-num">{{ item.value }}</div>
          <div class="stat-label">{{ item.label }}</div>
        </el-card>
      </el-col>
    </el-row>
    <!-- ECharts 折线图 -->
    <el-card style="margin-top:16px">
      <div ref="chartRef" style="height:300px"></div>
    </el-card>
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'
import * as echarts from 'echarts'
import { getStats, getTrend } from '@/api/dashboard'

const stats = ref([])
const chartRef = ref()

onMounted(async () => {
  const statsData = await getStats()
  stats.value = [
    { label: '博客总数', value: statsData.blogTotal },
    { label: '今日访问', value: statsData.todayViews },
    { label: '评论总数', value: statsData.commentTotal },
    { label: '用户总数', value: statsData.userTotal },
  ]

  const trendData = await getTrend()
  const chart = echarts.init(chartRef.value)
  chart.setOption({
    xAxis: { type: 'category', data: trendData.dates },
    yAxis: { type: 'value' },
    series: [{ data: trendData.views, type: 'line', smooth: true }],
    tooltip: { trigger: 'axis' }
  })
})
</script>
```

**安装 ECharts：**
```bash
npm install echarts
```

### 2. 博客列表（BlogListView.vue）

关键功能：
- `el-table` 展示博客列表，支持 `status`（状态下拉）、`categoryId`（分类下拉）、`keyword`（关键词）筛选
- 状态列使用 `el-tag` 展示：草稿（info）/ 已发布（success）/ 已下线（warning）
- 操作列：编辑（跳转 `/blog/edit/:id`）、置顶切换、发布/下线、删除（二次确认）

```javascript
// 状态操作
async function toggleStatus(row) {
  const newStatus = row.status === 1 ? 2 : 1
  await updateBlogStatus(row.id, newStatus)
  row.status = newStatus
  ElMessage.success(newStatus === 1 ? '已发布' : '已下线')
}
```

### 3. 博客编辑（BlogEditView.vue）— 重点页面

```vue
<template>
  <div class="blog-edit">
    <el-form :model="form" label-width="80px">
      <el-form-item label="标题">
        <el-input v-model="form.title" placeholder="博客标题" />
      </el-form-item>
      <el-form-item label="封面图">
        <el-upload action="#" :http-request="uploadCover" :show-file-list="false">
          <img v-if="form.coverImg" :src="form.coverImg" class="cover-preview" />
          <el-button v-else>上传封面</el-button>
        </el-upload>
      </el-form-item>
      <el-form-item label="分类">
        <el-select v-model="form.categoryId" placeholder="选择分类">
          <el-option v-for="c in categories" :key="c.id" :label="c.name" :value="c.id" />
        </el-select>
      </el-form-item>
      <el-form-item label="标签">
        <el-select v-model="form.tagIds" multiple placeholder="选择标签">
          <el-option v-for="t in tags" :key="t.id" :label="t.name" :value="t.id" />
        </el-select>
      </el-form-item>
      <el-form-item label="摘要">
        <el-input v-model="form.summary" type="textarea" :rows="2" placeholder="留空则自动截取正文前150字" />
      </el-form-item>
      <!-- Markdown 编辑器 -->
      <el-form-item label="正文">
        <MdEditor v-model="form.content" :toolbars="toolbars" @onUploadImg="handleUploadImg" />
      </el-form-item>
      <el-form-item>
        <el-button @click="save(0)">保存草稿</el-button>
        <el-button type="primary" @click="save(1)">发布</el-button>
      </el-form-item>
    </el-form>
  </div>
</template>

<script setup>
import { MdEditor } from 'md-editor-v3'
import 'md-editor-v3/lib/style.css'
import { uploadFile } from '@/api/file'

// 编辑器内图片上传
async function handleUploadImg(files, callback) {
  const urls = await Promise.all(files.map(f => uploadFile(f)))
  callback(urls)
}

// 封面上传
async function uploadCover({ file }) {
  form.coverImg = await uploadFile(file)
}
</script>
```

### 4. 评论管理（CommentView.vue）

- 默认展示待审核评论（`status=0`）

### 5. 轮播图管理（BannerView.vue）

- 图片以卡片形式展示（`el-image` 预览）
- 支持调整 `sort_order`（直接输入数字保存）
- 启用/禁用开关（`el-switch`）
