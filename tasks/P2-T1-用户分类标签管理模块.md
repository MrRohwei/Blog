| PUT | `/admin/v1/auth/password` | 修改密码 |

### 关键实现

```java
// 新建用户：密码 BCrypt 加密
public void createUser(CreateUserRequest req) {
    if (userMapper.selectCount(new LambdaQueryWrapper<TUser>()
            .eq(TUser::getUsername, req.getUsername())) > 0) {
        throw new BusinessException(ErrorCode.USERNAME_DUPLICATE);
    }
    TUser user = new TUser();
    BeanUtils.copyProperties(req, user);
    user.setPassword(passwordEncoder.encode(req.getPassword()));
    userMapper.insert(user);
}

// 分页查询（支持用户名模糊搜索）
public PageResult<UserVO> listUsers(UserQueryRequest req) {
    Page<TUser> page = new Page<>(req.getPage(), req.getSize());
    LambdaQueryWrapper<TUser> wrapper = new LambdaQueryWrapper<TUser>()
        .like(StringUtils.hasText(req.getKeyword()), TUser::getUsername, req.getKeyword())
        .orderByDesc(TUser::getCreatedAt);
    IPage<TUser> result = userMapper.selectPage(page, wrapper);
    return PageResult.of(result.convert(UserVO::from));
}
```

### 验收标准

- [ ] `GET /admin/v1/users?page=1&size=10` 返回分页数据，密码字段不返回
- [ ] `POST /admin/v1/users` 创建用户，重复用户名返回 `code: 1004`
- [ ] `PUT /admin/v1/users/{id}/status` 修改状态后，对应用户登录返回"已禁用"错误
- [ ] 删除用户后数据库记录消失（或软删除标记）

## 模块二：分类管理

### 接口清单

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/categories` | 前台：分类列表（含博客数） |
| GET | `/admin/v1/categories` | 后台：分页列表 |
| POST | `/admin/v1/categories` | 新建分类 |
| PUT | `/admin/v1/categories/{id}` | 编辑分类 |
| DELETE | `/admin/v1/categories/{id}` | 删除分类 |

### 关键实现

```java
// 前台分类列表：统计每个分类下已发布的博客数
public List<CategoryVO> listCategoriesWithCount() {
    return categoryMapper.selectCategoriesWithBlogCount();
}
```

```xml
<!-- CategoryMapper.xml -->
<select id="selectCategoriesWithBlogCount" resultType="CategoryVO">
    SELECT c.id, c.name, c.description, c.sort_order,
           COUNT(b.id) AS blog_count
    FROM t_category c
    LEFT JOIN t_blog b ON c.id = b.category_id AND b.status = 1
    WHERE c.status = 1
    GROUP BY c.id
    ORDER BY c.sort_order ASC
</select>
```

### 验收标准

- [ ] `GET /api/v1/categories` 返回启用中的分类，每项包含 `blogCount` 字段
- [ ] 分类名重复时新建返回错误提示
- [ ] 删除分类时若该分类下有博客，返回友好错误（不允许直接删除）
