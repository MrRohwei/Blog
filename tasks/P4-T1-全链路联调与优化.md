#### 全流程业务测试

1. 后台登录 → 新建分类 → 新建标签
2. 新建博客（设置分类、标签、封面、Markdown 正文）→ 发布
3. 前台首页刷新 → 博客出现在列表
4. 前台点击博客 → 详情页正确展示（Markdown 渲染 + TOC）
5. 详情页点赞 → 浏览量 +1
6. 前台提交评论 → 后台审核通过 → 前台刷新可见评论
7. 后台将博客设为下线 → 前台访问返回 404
8. 后台登出 → 原 Token 访问接口返回 401

### 文件上传安全

```java
// 限制文件类型和大小
@PostMapping("/upload")
public Result<String> upload(@RequestParam MultipartFile file) {
    // 文件大小限制（5MB）
    if (file.getSize() > 5 * 1024 * 1024) {
        throw new BusinessException(400, "文件大小不能超过 5MB");
    }
    // 文件类型白名单
    String contentType = file.getContentType();
    if (!List.of("image/jpeg", "image/png", "image/gif", "image/webp")
            .contains(contentType)) {
        throw new BusinessException(400, "仅支持上传 JPG/PNG/GIF/WebP 图片");
    }
    return Result.success(ossService.upload(file));
}
```

### application.yml 上传限制

```yaml

  servlet:
    multipart:
      max-file-size: 10MB
      max-request-size: 10MB
```
