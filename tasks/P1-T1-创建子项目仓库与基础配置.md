## 任务目标

建立 Voidpen 项目的三个子应用仓库，完成本地开发环境初始化，确保三个项目均可在本地正常启动。

---

## 详细步骤

### 1. 创建 Git 仓库

```bash
# 推荐结构（monorepo 或三个独立仓库均可）
voidpen/
├── voidpen-server/     # Spring Boot 后端
├── voidpen-frontend/   # Vue 3 前台
└── voidpen-admin/      # Vue 3 后台
```

- 在 GitHub / Gitee 创建仓库（建议设为 Private）
- 每个仓库初始化 `README.md` 和 `.gitignore`

### 2. 初始化 voidpen-server（Spring Boot）

使用 [Spring Initializr](https://start.spring.io) 生成项目，配置如下：

| 配置项 | 值 |
|--------|----|
| Project | Maven |
| Language | Java |
| Spring Boot | 3.x（最新稳定版） |
| Java | 17 |
| Group | com.voidpen |
| Artifact | voidpen-server |

**引入依赖（Initializr 勾选）：**
- Spring Web
- Spring Security
- MyBatis-Plus（手动在 pom.xml 添加）
- MySQL Driver
- Spring Data Redis
- Validation
- Lombok

**验证：** 运行 `VoidpenServerApplication`，控制台输出 `Started VoidpenServerApplication` 即为成功。

### 3. 初始化 voidpen-frontend（Vue 3 前台）

```bash
npm create vite@latest voidpen-frontend -- --template vue
cd voidpen-frontend
npm install
npm install vue-router@4 pinia axios
npm install marked highlight.js dompurify
npm run dev
```

**验证：** 浏览器访问 `http://localhost:5173`，显示 Vite + Vue 默认页面。

### 4. 初始化 voidpen-admin（Vue 3 后台）

```bash
npm create vite@latest voidpen-admin -- --template vue
cd voidpen-admin
npm install
npm install vue-router@4 pinia axios
npm install element-plus @element-plus/icons-vue
npm install md-editor-v3
npm run dev
```

**验证：** 浏览器访问 `http://localhost:5174`，显示 Vite + Vue 默认页面。

### 5. 配置 .gitignore

**voidpen-server `.gitignore` 关键内容：**
```
target/
*.class
*.jar
.env
application-prod.yml
.idea/
*.iml
```

**前端项目 `.gitignore` 关键内容：**
```
node_modules/
dist/
.env.local
.env.production
```

### 6. 提交初始代码

```bash
git add .
git commit -m "feat: 初始化项目骨架"
git push origin main
```

## 验收标准

- [ ] 三个仓库均已推送至远程，可正常 `git clone`
- [ ] `voidpen-server` 执行 `mvn spring-boot:run` 无报错启动，端口 8080
- [ ] `voidpen-frontend` 执行 `npm run dev` 正常访问 localhost:5173
- [ ] `voidpen-admin` 执行 `npm run dev` 正常访问 localhost:5174
- [ ] 三个项目均有合理的 `.gitignore`，`node_modules/`、`target/`、`.env` 等不进入版本控制
- [ ] 每个项目 `README.md` 包含本地启动步骤说明
