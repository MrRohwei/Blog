## 详细步骤

### 1. 安装 JDK 17

```bash
sudo apt update
sudo apt install -y openjdk-17-jdk

# 验证
java -version
# 输出：openjdk version "17.x.x"
```

### 2. 安装 MySQL 8.0

```bash
sudo apt install -y mysql-server
sudo systemctl start mysql
sudo systemctl enable mysql

# 初始化安全配置
sudo mysql_secure_installation

# 创建数据库和用户
sudo mysql -u root -p
```

```sql
CREATE DATABASE voidpen DEFAULT CHARACTER SET utf8mb4;
CREATE USER 'voidpen'@'localhost' IDENTIFIED BY 'your_strong_password';
GRANT ALL PRIVILEGES ON voidpen.* TO 'voidpen'@'localhost';
FLUSH PRIVILEGES;

-- 导入初始化 SQL
SOURCE /opt/voidpen/sql/init.sql;
```

### 3. 安装 Redis 7.x

```bash
sudo apt install -y redis-server

# 配置密码和持久化
sudo vim /etc/redis/redis.conf
```

**redis.conf 关键配置：**
```
# 设置密码
requirepass your_redis_password

# 开启 AOF 持久化
appendonly yes

# 只监听本地（不对外暴露）
bind 127.0.0.1
```

```bash
sudo systemctl restart redis
sudo systemctl enable redis

# 验证
redis-cli -a your_redis_password ping
# 输出：PONG
```

### 4. 安装 Nginx

```bash
sudo apt install -y nginx
sudo systemctl start nginx
sudo systemctl enable nginx

# 验证：访问服务器 IP，看到 Nginx 欢迎页
```

### 5. 配置 application-prod.yml

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/voidpen?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: ${DB_USERNAME}
    password: ${DB_PASSWORD}
    driver-class-name: com.mysql.cj.jdbc.Driver
  data:
    redis:
      host: localhost
      port: 6379
      password: ${REDIS_PASSWORD}
      database: 0

voidpen:
  jwt:
    secret: ${JWT_SECRET}
    expire: 7200
  oss:
    access-key-id: ${OSS_KEY_ID}
    access-key-secret: ${OSS_KEY_SECRET}
    endpoint: ${OSS_ENDPOINT}
    bucket: ${OSS_BUCKET}

# 生产环境关闭接口文档
knife4j:
  enable: false
```

### 6. 创建服务器目录结构

```bash
sudo mkdir -p /opt/voidpen/{server,frontend,admin,sql,logs}
sudo chown -R $USER:$USER /opt/voidpen
```

### 7. 编写启停脚本

```bash
# /opt/voidpen/server/start.sh
#!/bin/bash
export DB_USERNAME=voidpen
export DB_PASSWORD=your_db_password
export REDIS_PASSWORD=your_redis_password
export JWT_SECRET=your-256-bit-secret-key-for-production
export OSS_KEY_ID=your_oss_key
export OSS_KEY_SECRET=your_oss_secret
export OSS_ENDPOINT=oss-cn-hangzhou.aliyuncs.com
export OSS_BUCKET=voidpen-bucket

nohup java -jar /opt/voidpen/server/voidpen-server.jar \
  --spring.profiles.active=prod \
  --server.port=8080 \
  > /opt/voidpen/logs/app.log 2>&1 &

echo "Voidpen Server started, PID: $!"
echo $! > /opt/voidpen/server/app.pid
```

```bash
# /opt/voidpen/server/stop.sh
#!/bin/bash
PID=$(cat /opt/voidpen/server/app.pid)
if [ -z "$PID" ]; then
  echo "No PID file found"
else
  kill $PID
  echo "Stopped PID: $PID"
  rm /opt/voidpen/server/app.pid
fi
```

```bash
chmod +x /opt/voidpen/server/start.sh
chmod +x /opt/voidpen/server/stop.sh
```

## 验收标准

- [ ] `java -version` 输出 OpenJDK 17
- [ ] MySQL 服务运行中，`voidpen` 数据库存在，8 张表已创建
- [ ] Redis 服务运行中，`redis-cli ping` 返回 PONG
- [ ] Nginx 服务运行中，访问服务器公网 IP 显示 Nginx 欢迎页
- [ ] `application-prod.yml` 不含任何明文密码（全部使用环境变量占位符）
- [ ] `start.sh` 脚本可正常启动服务，`tail -f /opt/voidpen/logs/app.log` 无启动错误
- [ ] 后端服务启动后，`curl http://localhost:8080/admin/v1/auth/login -X POST` 有响应（即使返回 400 也证明服务正常）
